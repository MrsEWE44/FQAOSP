# FQAOSP是一个适用于类原生的工具箱，它可以帮你完成一些root的操作

比如应用分身、挂载手机上的镜像文件给电脑用，或者反编译apk文件，或者干掉烦人的后台进程等等，它都可以实现，只要你愿意，它都能做到。

虽然现在是起步阶段，但会尽快添加新功能进来，让它更加丰富强大。
